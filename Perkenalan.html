HTML (HyperText Markup Language)

HTML adalah bahasa dasar yang digunakan untuk membuat struktur dan isi dari sebuah halaman web. Dengan HTML, kita dapat menentukan teks, gambar, tautan, tabel, form, dan berbagai elemen lainnya.

CSS (Cascading Style Sheets)

CSS digunakan untuk memberikan gaya pada halaman web — seperti warna, tata letak, font, dan animasi. Dengan CSS, tampilan web menjadi lebih menarik dan responsif di berbagai perangkat.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas pada halaman web. Contohnya seperti tombol yang bisa diklik, efek animasi, atau validasi form.

Tailwind CSS

Tailwind CSS adalah framework CSS modern berbasis utility-first. Dengan Tailwind, kita dapat mendesain tampilan langsung di dalam HTML tanpa perlu membuat banyak file CSS.
